# üë®‚Äçüíº Admin User Stories

### **Access & User Management**
- As an **Admin**, I want to create user accounts, so that new doctors and patients can access the system.
- As an **Admin**, I want to assign roles (Admin/Doctor/Patient), so that users have the correct permissions.
- As an **Admin**, I want to deactivate or delete accounts, so that I can maintain system security.

### **System Oversight**
- As an **Admin**, I want to view all appointments, so that I can monitor clinic activity.
- As an **Admin**, I want to view system logs, so that I can track changes and detect issues.
- As an **Admin**, I want to manage clinic-wide settings, so that the system remains up to date.

---

# üßë‚Äç‚öïÔ∏è Doctor User Stories

### **Appointment Management**
- As a **Doctor**, I want to view my upcoming appointments, so that I can prepare for patient visits.
- As a **Doctor**, I want to update my availability, so that patients can book appointments at the right times.
- As a **Doctor**, I want to cancel or reschedule appointments, so that I can manage my schedule effectively.

### **Patient Interaction**
- As a **Doctor**, I want to view patient medical history, so that I can provide informed care.
- As a **Doctor**, I want to add notes after each appointment, so that patient records remain accurate.
- As a **Doctor**, I want to create prescriptions, so that patients receive the correct medication.

# üßë‚Äçü¶Ω Patient User Stories

### **Appointment Booking**
- As a **Patient**, I want to view available doctors, so that I can choose the right specialist.
- As a **Patient**, I want to book an appointment, so that I can receive medical care.
- As a **Patient**, I want to cancel or reschedule my appointment, so that I can manage my schedule.

### **Medical Records**
- As a **Patient**, I want to view my past appointments, so that I can track my medical history.
- As a **Patient**, I want to view my prescriptions, so that I can access medication details.
- As a **Patient**, I want to update my personal information, so that my profile stays accurate.
